Commands

Marco'S offers a wide range of commands. It has Nine Command Categories. Click the category below to view it's commands




Configuration Commands

/ban-setting

Sets the ban system

/kick-setting

Sets the kick system

/registry-role

The registration system sets the roles

/registry-status

Determines the status of the recording system

/autorole-setting

Sets the automatic role system

/channel-log

Sets the channel log channel

/message-log

Sets the message log channel

/mod-log

The moderator sets the log channel

/role-log

Sets the role log channel

/incoming-outgoing

Sets the Incoming-Outgoing channel



Moderation Commands

/ban

Bans the user from the server

/unban

Removes the user's ban

/kick

Kicks the user out of the server

/nuke

Creates the channel from the beginning

/purge

Deletes messages

/registry

Registers the user


General Commands

/avatar

Returns the user's avatar

/ping

You display the delay values of the system

/info

Do you need help?

/settings

You can see the server settings
